Welcome to the Code Institute Full Stack Software Development Resource Pack! Like many of you, I am relatively new to the world of digital innovation, and through my time on Slack have come across some very useful Resources, Tools & Additional supports that have helped me in my journey to date.
My name is Anthony and as recently appointed student lead for User Centric Frontend Development, I decided to create a "webspace" dedicated to these supplementary solutions available to me, and now to you!

Resources:

Click on any of the below headers to be taken directly to the resource.

  • W3Schools

    W3 is an educational website for learning programming languages & frameworks, such as HTML, CSS & Bootstrap (and many more). All of which you will be using through UCFD!

  • MDN Mozilla

    MDN Web Docs is an evolving learning platform for Web tech & the software that powers the Web, such as HTML, CSS & Javascript to name a few.

  • Anna Greaves' Bootstrap Grid Tutorial

    Anna Greaves, a current student on the course we are all on and my predecessor as UCFD Lead, has created an incredible resource on the Bootstrap Grid. I really think Code Institute should consider putting this video in the curriculum, as I know it has helped not only me, but many MANY other students in their understanding of how to navigate the Grid!

  • Traversy Media Youtube Channel

    For the sake of this webspace, I couldn't possibly link the thousands of videos Brad from Traversy Media has made that could possibly help us in our journey, but I have linked his Crash Course on CSS3 video for you all to sink your teeth into. (P.S: There is also a HTML5 Crash course video too.)

  • Design Course Youtube Channel

    Take a dive into the mind of a Frontend, state of the art UI Designer, again I couldn't possible link all of the useful videos he has made, but I have linked Gary's video on the 2019 Frontend Developer Crash Course (HTML & CSS).

  • Markdown Cheat Sheet

    This document was a godsend for me when I was writing my README.md file for my 1st milestone and I'm sure I'll be referencing it again for all the rest of my Milestone projects. The link will take you to Adam Pritchard's Github and the repo where this cheatsheet is housed!

  • Jon Duckett's Reference Books

    Jon Duckett has created an inspirational set of reference material for anyone from the tech-savvy experienced coder to anyone with no coding experience. I bought these within my first week and have read them extensively! From HTML & CSS to Javascript & jQuery you are sure to use these throughout your entire course.

  • Bootstrap Documentation

    If you can think of an element, Bootstrap probably has it, from responsive imagery to dynamic collapsable elements. The documentation is easy to follow and you will be using Bootstrap to structure your Milestone projects to be responsive!

  • Font Awesome Iconography Documentation

    Welcome to the world of Iconography! Font Awesome has a host of free icons that you can use to uplift your html elements to a more unique and eye-popping appeal. The documentation deals with start-up, linking to the website and the icons and how to style them to fit the theme/palette of your website!

  • Elements of User Experience

    This document write up was found by a fellow student David 'H' Hubber, it is written by Jesse James Garrett and is only one chapter of an even bigger resource. It details the importance of each of the five planes of User Experience.

  • User Stories Examples

    A quick google search for User Story Examples returned thousands of search results in seconds, the link is on "How to create a good user story". For the smallest piece of work you will create during your README.md write up, it can sometimes be the hardest, give this a good run through and you should be ok to create your own!

  • How to make a Kick-Ass first Milestone!

    These documents, of which there are 3, are the go to Gold Star in terms of useful resources. Once again a special thanks to Anna Greaves, my predecessor, on creating such a benchmark of knowledge that we can all use and learn from when we feel a bit lost regarding the direction to take with our projects!

  • Learn how to git it

    Do you "Git it"? If not, then why not get ahead of the curve and "Learn how to Git it"! From the masterful mind of one of the Course Mentors Simen Daehlin, this will surely set you on the right path! This resource is often used in tandem with Simen's great Slack Calls on the topic he hosts that has helped not only me, but countless others! I highly recommend.

Tools:

Click on any of the below card headers to be taken directly to the tool!

Adobe Color

The Adobe Color service helps you choose harmonious and appealing color combinations for your design projects, it even has an upload image element that takes your image through it's intuitive system and picks the color scheme for you!

CSS Matic

The CSS Matic helps you to generate a gradient color of your structure, border radius towards a circle or a rectangle, background noise texture and box shadows using CSS and providing you all the relevant toolkits to work on the different browsers!

Color Picker

This simple addition to Chrome will make your life so much easier when trying to find the color a particular site is using, made by Kepi on Github and the Color Picker that I have used for quite some time now.

HTML Validator

The gold star in terms of trying to find that missing `.` or `-` in your html markup. Any developer worth their salt, will use this tool on a regular basis to validate their code and make sure every div is closed and every element is structured correctly!

Chrome DevTools

Ok, so I obviously can't link you to the DevTools, but I have linked a video from Traversy Media, which is a tutorial Crash Course on how to open and use! I 100% recommend you get used to this Tool, as it is every developer's best friend.

CSS Validator

As with the HTML Validator, this CSS version ensure that you have accounted for every possible closing curly brace, every `,` comma seperated when needed. Another one of those essential tools for all developers, maintaining that tool of the trade mantra!

Boostnote

A truly distraction free, live preview, what you see is what you get Markdown Editor. Install this to your local device for a simple and effective README editor.

CSS Autoprefixer

Take your CSS code blocks, and parse them through this incredible tool, which digests those blocks and returns the appropriate vendor prefixes needed for your code to work in ALL browsers, anything from animations to flexbox and eveything in between.

Diffchecker

This tool enables you to compare text differences between two text files, and it also works on html. Scenario: you have source code, but when you implement it into your code, it just doesn't look right, that's where Diffchecker comes in! Let it tell you!

TinyJPG.com

Take your imagery to the next level of compression with TinyJPG. Upload your image, let the analyser do it's job and watch as the encoder compresses & creates an optimally compressed version of that file for you to download.

Balsamiq

This link will bring you back to Slack, where you can follow the post from Niel_ci to download and get this wireframing tool completely for Free! With Balsamiq, anyone can design and plan great website for all viewport widths.

Pexels

Pexels provides free stock photos, which can be used for commercial design purposes without the hassle of copyright infringement. Very easy site to navigate and with thousand of photos, don't get lost!! Just click and download to your local device.

ColorSpace

Another great tool on finding that perfect Color palette. Does your website logo contain one dominant color? Cool, just type in the hexcode into ColorSpace and let it tell you the perfect combination of colors for your project.

CodePen.io

Welcome to the world of online's easiest IDE!! CodePen.io.

Colormind

Colormind is a color scheme generator that uses deep learning. It can learn color styles from photographs, movies, and popular art.

WebAIM

Color contrast checker.

Unicorn Revealer

Do you have overflow on your website and want to try find it. Sparkle some 🦄🌈on your website and find where it is. As a web developer it can be hard to inspect and find out where overlow is located. You can sprinke some 🦄🌈 on the website and it will turn all black and pink, this will show the borders and make it easier to debug. It is as simple as right click a website Sparkle it on once done right click and remove the sparkle.

Additional:

Supplementary info! Helpful tips, advice & quotes from Slack users.

A hot topic right now is "What Editor can we use?". Recently Code Institute students had to migrate from Cloud9, (an online IDE), to AWS-Cloud9, (Cloud9 assumed by Amazon services). With this migration, it brought a host of new issues, learning curves and new questions such as the one above. For the record, although it is recommended that students use the AWS-C9 IDE, it is important to note that this is not mandatory.
Even before the migration, I myself opted to use an editor called PyCharm, which was recommended to me by my mentor. I followed along to the course material adjusting as needed, and have been using it ever since, (my entire Milestone 1 was coded in PyCharm).
Below you will find a short list detailing some of the IDE's I've seen people talk about using in the Slack App:

  • Visual Studio Code

    Free. Built on open source. Runs everywhere. Probably the most preferred in our community right now.

  • Brackets

    A modern, open source text editor that understands web design.

  • Atom

    A hackable text editor for the 21st Century, with built in Git integration.

  • PyCharm

    PyCharm is designed by programmers, for programmers, to provide all the tools you need for productive development (with high focus on Python).

  • Notepad++

    Notepad++: a free source code editor which supports several programming languages running under the MS Windows environment.
